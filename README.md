# Customer-Analytics-Dashboard
This dashboard provides a comprehensive view of the distribution of customers across gender, generations, level of income, occupations, loyalty segments, and overall growth index
## Project Overview
This project focuses on analyzing customer demographics, behavior, and value segmentation to support data-driven decision-making in marketing and customer relationship management.By combining demographic and behavioral insights, the analysis helps identify key customer groups, track business growth, and uncover opportunities for targeted engagement and revenue optimization.
