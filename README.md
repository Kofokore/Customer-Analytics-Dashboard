# Customer-Analytics-Dashboard
This dashboard provides a comprehensive view of the distribution of customers across gender, generations, level of income, occupations, loyalty segments, and overall growth index
## Project Overview
This project focuses on analyzing customer behavior, demographics, and value segmentation to support data-driven decision-making in customer relationship management and marketing. By combining demographic and behavioral insights, this analysis helps to identify key customer groups, track business growth, and discover opportunities for targeted engagement and maximum revenue optimization.
